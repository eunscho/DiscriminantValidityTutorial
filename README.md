It contains various data and codes for the tutorial on discriminant validity.


*DATA FILES

txt format, used in Mplus: ACSICovData.dat, ACSICovData.txt



*Software syntax FILES

AMOS: ACSITutorial.amw

LISREL: ACSI.ls8

Mplus: ACSITutorial.inp

R: ACSIdemo.R

STATA: ACSIdemo.do
